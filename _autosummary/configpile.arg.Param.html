Param¶

class Param[source]¶

Bases: configpile.arg.Arg, Generic[configpile.arg._Value]

Describes an argument holding a value of a given type

You’ll want to construct parameters using the static methods available:

  • store() to create a parameter that keeps the last value provided

  • append() to create a parameter that collects all the provided value into a sequence. Note that the parsed value must already be a sequence; for that, use the method as_sequence_of_one() on the ParamType instance.

For the two constructions above, you can construct positional parameters by setting the Param.positional field.

  • config() returns a parameter that parses configuration files.
